# 프로젝트 설정


## 개요
선택한 프로젝트의 설정 정보를 확인하고 수정할 수 있습니다.  
(프로젝트 설정 기능을 수행하기 위해 프로젝트 관리자 또는 시스템 관리자 권한이 필요합니다.)


## 기능 구성
- 프로젝트 정보 확인 및 수정  
- 접근 권한 설정  
- 이미지 메타 데이터 설정  
- 어노테이션 타입 설정  
- 모델 타입 설정  
- 모델셋 타입 설정  
- 모델셋 배포 관리 설정  
- 불량 유형 설정  
- 서비스 위치 설정  
- 시스템 설정
