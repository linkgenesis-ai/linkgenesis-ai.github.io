# 레이블링

## 개요
데이터셋에 포함된 이미지에 어노테이션을 작성하는 도구와 검색 기능을 제공합니다.

## 기능 구성

- 이미지 조회  
  필터를 이용하여 데이터셋에 포함된 이미지를 조회할 수 있습니다.
- 레이블링  
  선택한 이미지에 어노테이션을 작성할 수 있습니다.
- 데이터셋 생성  
  조회된 이미지로 신규 데이터셋을 생성하거나 기존 데이터셋에 추가할 수 있습니다.
