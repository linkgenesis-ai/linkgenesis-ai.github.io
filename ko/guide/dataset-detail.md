# 데이터셋 상세 정보

![데이터셋 상세 정보](/public/ko/data/dataset-detail.png)


## 개요
데이터셋의 상세 구성 내용을 확인하고, 데이터셋 연계 기능을 수행할 수 있습니다.

- 기본 정보 : 데이터셋의 기본 구성 정보
  - 복제된 데이터셋인 경우 `복제 원본` 정보 표시
- 포함된 이미지 타입 : 데이터셋에 포함된 이미지 유형
- 레이블 수 : 데이터셋에 작성된 어노테이션 유형 및 갯수


## 기능
- 데이터셋 레이블링
- 데이터셋 복제
- 데이터셋 삭제
- 데이터셋 학습
- 데이터셋 내보내기
- 데이터셋 메타 업데이트
- 데이터셋 이름, 설명 수정
