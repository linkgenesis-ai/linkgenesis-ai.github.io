# 학습 및 검증

## 개요
학습 및 검증 기능 소개

## 기능 구성
- 학습 커널  
  학습용 커널을 업로드하고 관리합니다.
- 모델 학습  
  학습용 모델을 업로드하고 모델 학습 작업을 관리합니다.
- 모델 검증  
  모델 검증 작업을 확인하고 검증 결과를 확인합니다.
- 활성 모델  
  추론에 사용할 수 있는 모델 목록을 확인하고 관리합니다.
