# 서비스 위치 설정

![서비스 위치 설정](/public/ko/project/project-settings-service-location.png)


## 개요
서비스 위치 정보를 관리할 수 있습니다.  
(외부 서비스 연동 기능을 사용할 경우에만 활성화)

- 코드 : 서비스 위치 코드
- 코드값 : 서비스 위치 코드값


※ 서비스 위치 코드와 코드값은 커스터마이징 작업을 거쳐 결정됩니다.


## 기능
- 서비스 위치 추가
  - "추가" 버튼을 클릭하여 서비스 위치 정보 항목을 추가할 수 있습니다.
