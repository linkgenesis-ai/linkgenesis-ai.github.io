# 데이터셋 관리

![데이터셋 관리](/public/ko/data/dataset-list.png)


## 개요
생성한 데이터셋의 목록을 확인하고 필터를 설정하여 조회할 수 있습니다.


## 기능
- 데이터 가져오기
- 데이터셋 상세 정보 확인하기

### 데이터 가져오기
`가져오기` 버튼을 클릭하여 [데이터 가져오기](./dataset-import) 기능을 실행합니다.

### 데이터셋 상세 정보 확인하기
조회된 데이터셋 목록의 항목을 클릭하면 [데이터셋 상세 정보](./dataset-detail.md) 화면으로 진입합니다.
